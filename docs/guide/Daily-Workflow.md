# Daily Workflow

학습 세션을 짧고 일관되게 유지하기 위한 권장 루틴.

## 시작 (5분)

1. `/next`로 우선순위 1-3개 확인
2. 오늘 목표 1개 선택
3. `/study plan <오늘 목표>` 기록

## 집중 세션 (20-40분)

1. 신규 학습이면 `/learn <skill> <topic>`
2. 복습이면 `/review <skill> [topic]`
3. 종료 직전 "정리"로 기록 반영

## 마감 (5분)

1. `/study done <완료한 내용>`
2. `/study log`
3. 다음 세션 시작점을 1줄로 메모

## 주간 루틴

- 주 1회 `/plan [goal]`으로 `docs/master-plan.md` 점검
- 필요 시 `/study-skill <skill>`로 커버리지 갭 보정

## 운영 팁

- 새 토픽 시작 전 `/next` 근거를 먼저 확인한다.
- 학습 세션과 복습 세션을 섞어 과부하를 줄인다.
- 세션이 길어지면 한 번 종료("정리") 후 재개한다.

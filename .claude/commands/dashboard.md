# /dashboard — 전체 학습 현황 스냅샷

당신은 사용자의 학습 진행 상태를 한눈에 보여주는 대시보드입니다.
`docs/` 디렉토리를 스캔하여 스킬별 진행률, 최근 활동, 복습 대기 현황을 출력합니다.

> **읽기 전용**: 이 커맨드는 어떤 파일도 생성/수정하지 않습니다.
> **빠른 실행**: 판단/추천 없이 사실만 보여줍니다. 추천이 필요하면 `/next`를 사용하세요.

---

## Phase 1: Data Collection (자동)

### 1-1. 스킬 디렉토리 스캔

`docs/` 하위의 모든 디렉토리를 스캔합니다. 각 디렉토리명이 스킬명입니다.

```
Glob: docs/*/
```

→ `SKILL_LIST`: 스킬명 배열 (예: `["react", "nextjs"]`)

### 1-2. 스킬별 데이터 수집

각 스킬에 대해 아래를 수집합니다:

#### plan.md 파싱

`docs/{skill}/plan.md`가 존재하면:

1. **전체 토픽 수**: `### Topic N:` 패턴 카운트
2. **완료 토픽**: 체크리스트 3개(`소스 학습 완료`, `docs 교차 확인`, `skill 검증/개선`) 중 `[x]`가 1개 이상인 토픽
3. **커버율**: Coverage Analysis 테이블의 `커버율: N/M 모듈 (P%)` 파싱
4. **Phase 분포**: Phase 1/2/3별 토픽 수

plan.md가 없으면: `"플랜 미생성"` 표시

#### 학습 기록 파일 스캔

```
Glob: docs/{skill}/*.md (plan.md, *-quiz.md, *-meta.md 제외)
```

각 파일에서:
1. **파일명** → 토픽명 (하이픈 → 공백, Title Case)
2. **마지막 세션 날짜**: 파일 내 `## {날짜}` 패턴 중 마지막
3. **진행 상태**: 가장 최근 세션의 학습 로드맵에서 `[x]`/`[ ]` 카운트
4. **완료 여부**: 모든 마이크로 스텝이 `[x]`이면 완료

#### 복습 메타 스캔

```
Glob: docs/{skill}/*-meta.md
```

각 메타 파일에서:
1. **다음복습 날짜**: "개념별 상태" 테이블의 `다음복습` 컬럼 파싱
2. **기한 경과 개념 수**: 오늘 날짜 이전인 다음복습 날짜 카운트
3. **졸업 개념 수**: 상태가 `졸업`인 행 수

→ 각 스킬별로 `SKILL_DATA` 객체에 저장

### 1-3. 마스터 플랜 확인

`docs/master-plan.md`가 존재하면 마지막 갱신일을 파싱합니다.

## Phase 2: Dashboard Rendering (자동)

수집한 데이터를 아래 형식으로 출력합니다.

### 출력 형식

```markdown
## 학습 대시보드

> {오늘 날짜}

### 전체 요약

| 지표 | 값 |
|------|-----|
| 학습 중인 스킬 | {SKILL_LIST 수}개 |
| 전체 토픽 (plan 기준) | {합산 토픽 수}개 |
| 학습 완료 토픽 | {합산 완료 토픽 수}개 |
| 전체 진행률 | {바 그래프} {퍼센트}% |
| 복습 대기 개념 | {합산 기한 경과 수}개 |
| 마스터 플랜 | {있음 (갱신일) / 미생성} |

---

### 스킬별 현황

#### {skill-name}

| 지표 | 값 |
|------|-----|
| 진행률 | {바 그래프} {완료}/{전체} topics ({퍼센트}%) |
| 커버율 (plan 기준) | {커버 모듈}/{전체 모듈} ({퍼센트}%) |
| 최근 활동 | {마지막 세션 날짜} — {토픽명} |
| 복습 대기 | {기한 경과 개념 수}개 ({복습 대기 토픽명 목록}) |
| 졸업 개념 | {졸업 수}개 |

진행중 토픽:
{미완료 토픽 목록 — 토픽명 + Step 진행 상태}

{스킬별로 반복}

---

### 최근 활동 타임라인

| 날짜 | 스킬 | 토픽 | 상태 |
|------|------|------|------|
| {날짜} | {skill} | {topic} | {진행중 Step N/M / 완료} |

{최근 7개 세션, 날짜 역순}
```

### 바 그래프 생성 규칙

10칸 기준 블록 문자 사용:

```
0%:   ░░░░░░░░░░
30%:  ███░░░░░░░
75%:  ████████░░
100%: ██████████
```

### 빈 데이터 처리

- 스킬이 0개: "학습 기록이 없습니다. `/study-skill <skill-name>`으로 시작하세요."
- plan.md 없는 스킬: "플랜 미생성 — `/study-skill {skill}`로 생성"
- meta.md 없는 스킬: 복습 대기 `"-"` 표시
- 마스터 플랜 없음: "미생성 — `/plan`으로 생성"

## 주의사항

- **읽기 전용**: 어떤 파일도 생성/수정하지 않습니다.
- **판단 없음**: 추천, 우선순위 제안, 코멘트를 하지 않습니다. 사실만 보여줍니다.
- **빠른 실행**: 소스 코드(`ref/`)를 읽지 않습니다. `docs/` 파일만 스캔합니다.
- **추천 필요 시**: `/next`를 안내합니다.
- **플랜 필요 시**: `/plan`을 안내합니다.
